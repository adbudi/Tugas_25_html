Mengenal Semantic Element Di HTML 5

  • Articel

    • Ketika kita ingin membuat layout untuk menampilkan block artikel - artikel yang ada di website kita, tag artikel bisa kita gunakan dengan element header dan footer, Karena taq article memiliki type block, artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.


  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layput utam. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel. Karena aside memiliki type / sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
